# Viva Demo Seed Data

Run from `Users Mobile App`:

```powershell
.\tool\seed_viva_demo_supabase.ps1
```

The script populates Supabase with:

- 24 Sinhala/Tamil mobile users with profile-photo URLs.
- 20 keyed jobs with open, in-progress, and completed statuses.
- Applications, accepted workers, payments, reviews, chats, SMS logs, pending approvals, and volunteers.
